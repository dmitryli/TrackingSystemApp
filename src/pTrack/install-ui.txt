Installation instructions for pTrack UI.

1. Download and install nodejs from https://nodejs.org/

2. Open cmd and run:
npm install --global gulp

3. cd into pTrack folder.

4. run:
npm install

5. when done run:
bower install

You should have all necessary components by now.


Build UI:

1. open cmd and cd into pTrack folder.

2. run:
gulp

This should build all ui app into pTrack/dist folder which should be the app distribution folder wher you point server to and run it.


Configure server to serve the UI files:

1. In pivotal server configuration add "external web module" with the mapping of root ("/") to pTrack/dist folder.

2. Run DB and server

3. Open browser to http://localhost:8080/
